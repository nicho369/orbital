================================================================================
                           CHALLENGES ENCOUNTERED
================================================================================

1. CONFIGURATION ISSUES
   Problem: ES Module compatibility with Jest
   Solution: Converted config files to .mjs format and updated module resolution

2. REACT ROUTER TESTING
   Problem: Nested router conflicts when testing App component
   Solution: Created isolated route testing approach without nesting routers

3. TEXTENCODER POLYFILL
   Problem: TextEncoder not available in Jest/Node environment
   Solution: Added polyfill in jest.setup.ts

4. FIREBASE MOCKING
   Problem: Complex Firebase authentication state management
   Solution: Comprehensive mocking of auth methods and state changes

5. API ENDPOINT DISCREPANCIES
   Problem: Test expectations didn't match actual API endpoints
   Solution: Verified actual endpoints and updated test expectations
